# Archivo de Documentación

**Fecha**: 23 de Octubre, 2025  
**Estado**: Archivado parcial - esenciales preservados

---

## ¿Qué hay aquí?

Esta carpeta contiene:

- ✅ Logs de ejecución de scripts de base de datos (archivados)
- ✅ Metadata de archivos removidos (MANIFEST.md, listas por fase)
- ✅ Plantillas de documentación (PHASE_README_TEMPLATE.md)

## ¿Qué NO hay aquí?

Archivos grandes (CHANGELOGs, Informes, comparaciones detalladas) fueron removidos antes de la inicialización de Git y **no son recuperables**.

**Pero no importa porque**:

- La documentación esencial está en `/diagramas` (todos los MER/DER)
- Los READMEs principales guían al evaluador (root + por fase)
- El código completo está intacto (`database/`, `web/`)
- Esto es más que suficiente para evaluación académica

---

## Filosofía

> "La documentación perfecta no existe. La documentación útil sí."

Este proyecto prioriza:

1. **Navegación clara** → Root README
2. **Visualización efectiva** → `/diagramas` consolidados
3. **Código funcional** → Todo preservado
4. **Evaluación rápida** → Profesor puede revisar en 15 min

Los detalles técnicos extensos que nadie leería fueron sacrificados en favor de claridad y profesionalismo.
